# ⚡ Usabilidade

A usabilidade é um dos pilares fundamentais da Interface Humano-Computador. Ela determina o quão fácil e intuitivo é um sistema para seus usuários reais, em contextos reais de uso.

> "A eficácia, a eficiência e a satisfação com que usuários específicos conseguem alcançar objetivos específicos em ambientes particulares."

---

## 🎯 Metas de Aprendizagem

- Compreender o conceito de usabilidade e sua importância
- Identificar os três pilares da usabilidade
- Entender que usabilidade é definida pelo usuário, não pelo desenvolvedor
- Aplicar princípios de usabilidade no design de sistemas

---

## 📌 Os Três Pilares da Usabilidade

| Pilar | Descrição |
|-------|-----------|
| **✅ Eficácia** | O usuário consegue completar a tarefa? O sistema faz o que deveria fazer? |
| **⚡ Eficiência** | Quanto esforço (tempo, cliques, passos) o usuário precisa para completar a tarefa? |
| **😊 Satisfação** | O usuário se sente confortável e feliz usando o sistema? A experiência é agradável? |

---

## 💡 Princípios Fundamentais

- ✓ **Usabilidade significa foco no usuário** – O sistema deve ser projetado para quem realmente vai usá-lo
- ✓ **Pessoas utilizam produtos para serem produtivas** – O software é uma ferramenta, não um fim em si mesmo
- ✓ **Usuários são pessoas ocupadas tentando realizar tarefas** – Eles não querem "aprender o sistema", querem resolver problemas
- ✓ **Usuários decidem quando o produto é fácil de usar** – Não adianta o desenvolvedor achar intuitivo; quem define é quem usa

---

## 🚀 Conclusão

A usabilidade não é um luxo, é um requisito essencial. Sistemas com boa usabilidade:

- Reduzem erros e retrabalho
- Aumentam a produtividade
- Geram maior satisfação e fidelização
- Diminuem custos com suporte e treinamento

Lembre-se: **fácil de usar não é opiniao, é resultado de teste com usuário real.**

# ♿ Acessibilidade Digital

A acessibilidade digital garante que **todas as pessoas**, independentemente de suas limitações físicas, sensoriais ou cognitivas, possam utilizar os sistemas computacionais.

> "Garantir que os sistemas computacionais sejam construídos com atenção à acessibilidade."

---

## 🎯 Metas de Aprendizagem

- Compreender o conceito de acessibilidade digital
- Identificar a importância de softwares inclusivos
- Conhecer ferramentas de acessibilidade para pessoas com deficiência visual
- Desenvolver sistemas que atendam a diferentes necessidades

---

## 🛠️ Softwares para Inclusão de Pessoas com Deficiência Visual

Existem diversas ferramentas que permitem que pessoas com deficiência visual interajam com sistemas computacionais. Esses softwares são chamados de **leitores de tela**:

| Software | Descrição |
|----------|-----------|
| **🔊 Jaws** | Um dos leitores de tela mais completos e utilizados no mundo, muito presente em ambientes corporativos |
| **👁️ Virtual Vision** | Leitor de tela brasileiro, desenvolvido para atender o público nacional com suporte em português |
| **🆓 NVDA** | Leitor de tela gratuito e de código aberto, amplamente utilizado por ser acessível a todos |
| **📢 Dosvox** | Sistema brasileiro que integra leitor de tela, programas específicos e jogos, focado na inclusão digital |

---

## 📌 Importância da Acessibilidade

- **Inclusão social**: Garante que pessoas com deficiência possam estudar, trabalhar e se divertir
- **Cumprimento legal**: Muitos países possuem leis que exigem acessibilidade digital
- **Ampliação de público**: Sistemas acessíveis alcançam mais usuários
- **Qualidade para todos**: Recursos de acessibilidade frequentemente melhoram a experiência de todos os usuários

---

## 🚀 Conclusão

A acessibilidade digital não é apenas uma questão técnica, é uma **questão de direitos humanos**. Desenvolver sistemas acessíveis significa:

- Respeitar a diversidade humana
- Permitir que todos participem da sociedade digital
- Construir um mundo mais justo e inclusivo

> "A tecnologia mais poderosa é aquela que inclui."
